Тема: Розробка програм, що розгалужуються

Завдання:
Визначити, чи є ціле 6-значне число "щасливим квитком".
Щасливий квиток — це квиток, у якому сума першої половини чисел номера дорівнює сумі другої половини.

Основна частина:
Опис роботи основної функції:

Програма приймає ціле 6-значне число, перевіряє, чи є воно "щасливим квитком", і виводить результат.
Логіка роботи:
Вводиться 6-значне число.
Перевіряється, чи є число 6-значним. У разі невідповідності виводиться повідомлення про помилку.
Число розбивається на окремі цифри.
Обчислюються суми перших трьох і останніх трьох цифр.
Порівнюються отримані суми:
Якщо суми рівні — квиток "щасливий".
Інакше — квиток не є "щасливим".
Перелік вхідних даних:

Вхідне значення — ціле 6-значне число (тип int).
Перелік змінних:

ticketNumber — номер квитка (тип int).
digit1, digit2, ..., digit6 — окремі цифри числа (тип int).
firstHalfSum — сума перших трьох цифр (тип int).
secondHalfSum — сума останніх трьох цифр (тип int).
Дослідження результатів роботи програми:

При введенні 102300, програма визначає, що квиток є "щасливим".
Результат:
Квиток 102300 є щасливим!
При введенні 123456, програма визначає, що квиток не є "щасливим".
Результат:
Квиток 123456 не є щасливим.
Якщо введене число не є 6-значним (наприклад, 99999), програма виводить повідомлення про помилку:
Помилка: введене число не є 6-значним.

Контрольні питання:
Як працює умовний оператор if?
Оператор if виконує певний блок коду, якщо задана умова є істинною (true).
Який вираз називається складеним логічним? Наведіть приклади.
Складений логічний вираз містить кілька умов, об’єднаних операторами &&, ||, !.
Приклад: (a > 0 && b < 10).
Який оператор називають оператором множинного вибору? Наведіть приклад.
Оператор switch дозволяє вибирати один із кількох варіантів залежно від значення змінної.
Як працює оператор switch?
Оператор switch порівнює значення змінної зі значеннями case та виконує відповідний блок коду.
Як працює тернарний оператор? Наведіть приклад.
Тернарний оператор (? :) обчислює одне з двох значень залежно від умови:
int max = (a > b) ? a : b;
Коли умовний оператор називається вкладеним?
Оператор if називається вкладеним, якщо всередині одного if є інший.
Навіщо в операторі switch використовується оператор break?
Оператор break використовується для виходу з блоку switch після виконання відповідного блоку case, щоб уникнути виконання наступних блоків.
Чи можуть бути вкладеними оператори switch?
Так, оператори switch можуть бути вкладеними, але це слід використовувати з обережністю, щоб уникнути плутанини.
Чи можна замість оператора if використовувати тернарний оператор і навпаки?
Замість оператора if можна використовувати тернарний оператор, якщо потрібно обчислити значення однієї змінної.
Тернарний оператор не можна використовувати, якщо потрібно виконати кілька дій. У таких випадках потрібен оператор if.
