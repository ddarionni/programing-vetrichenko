Контрольні питання:
Як рядки представляються в мові програмування C?
У мові C рядки представляються як масив символів типу char, закінчений спеціальним символом \0 (нульовий символ), який позначає кінець рядка.
Приклад:
char str[] = "Hello";
Що таке “мірність масиву”? Скільки вимірів може мати масив?
Мірність масиву — це кількість індексів, необхідних для доступу до елементів масиву.
Масив може бути:
Одновимірним (наприклад, список).
Двовимірним (таблиця).
Багатовимірним (матриця, куб тощо).
У мові C кількість вимірів масиву обмежується лише об'ємом пам'яті.
Від чого залежить об’єм пам’яті, що необхідний для зберігання елементів масиву?
Обсяг пам’яті залежить від:
Кількості елементів масиву.
Розміру одного елемента (залежить від типу даних).
Формула:
Пам'ять = Кількість елементів × Розмір одного елемента
Як можна звернутися до елементу масиву?
До елемента масиву звертаються за допомогою його індекса. Індексація в C починається з 0.
Приклад:
int mas[5] = {1, 2, 3, 4, 5};
printf("%d", mas[2]); // Виведе 3
Як оголошуються масиви?
Масив оголошується вказанням типу даних, імені та кількості елементів у квадратних дужках.
Приклад:
int mas[10]; // Масив із 10 елементів
Дайте оцінку наступному оголошенню масиву:
int mas[] = { 1, 2, 3, 4, 5, 6, 7, 8 };
Оголошення коректне. Розмір масиву визначається автоматично на основі кількості ініціалізованих елементів (8 елементів).
При доступі до елементів двовимірного масиву укажіть, яке призначення першого та другого індексів?
Перший індекс вказує рядок.
Другий індекс вказує стовпець.
Приклад:
int mas[2][3] = {{1, 2, 3}, {4, 5, 6}};
printf("%d", mas[1][2]); // Виведе 6 (другий рядок, третій стовпець)
Дайте оцінку наступному оголошенню масиву:
int mas[][2] = {
    { 1, 2 },
    { 3, 4 },
    { 5, 6 },
    { 7, 8 }
};
Оголошення коректне. Перший розмір (кількість рядків) визначається автоматично за кількістю ініціалізованих підмасивів, а другий (кількість стовпців) задається явно.
В якому порядку зберігаються у пам’яті комп’ютера елементи двовимірного масиву?
Елементи двовимірного масиву зберігаються у пам'яті построчно (row-major order). Це означає, що всі елементи першого рядка зберігаються разом, потім елементи другого рядка і так далі.
Чим характеризуються діагональні елементи багатовимірних масивів? Скільки операторів циклу необхідно для їх обробки?
Діагональні елементи багатовимірного масиву мають однакові значення індексів у всіх вимірах (наприклад, mas[i][i]).
Для обробки діагональних елементів двовимірного масиву потрібен один цикл. Для багатовимірних масивів кількість циклів залежить від їх мірності.
Приклад для двовимірного масиву:
for (int i = 0; i < n; i++) {
    printf("%d ", mas[i][i]); // Виведення елементів головної діагоналі
}
