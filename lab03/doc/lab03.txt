Тема: Розробка лінійних програм

Завдання:
Підрахувати суму чисел у заданому діапазоні. Наприклад, при вхідних даних 50 та 52
повинно бути 50 + 51 + 52 = 153.

Основна частина:

Опис роботи основної функції:
Програма обчислює суму чисел у заданому діапазоні [start, end].
Сума обчислюється за допомогою циклу, який послідовно додає всі числа з діапазону до змінної sum.

Перелік вхідних даних:
- start — початок діапазону (ціле число, int).
- end — кінець діапазону (ціле число, int).

Перелік змінних:
- sum — результат обчислення суми чисел у діапазоні (ціле число, int).
- i — поточне значення ітерації циклу (ціле число, int).

У цій задачі всі дані вводятся користувачем, тому констант там немає, тільки змінні.

Дослідження результатів роботи програми:
1. При вхідних значеннях start = 50 та end = 52, обчислена сума:
   50 + 51 + 52 = 153
2. Для підтвердження коректності роботи програми:
   - У відлагоднику lldb встановлено точку зупинки на рядку return 0.
   - Виконано команду:
     print sum
   - Результат:
      153
   Як бачимо, отриманий результат підтверджує правильність виконання програми.

Контрольні питання:

1. З яких розділів складається програма мовою C?
   - Програма на C складається з наступних розділів:
     - Включення заголовкових файлів (#include).
     - Визначення констант і змінних.
     - Основна функція main (точка входу).
     - Додаткові функції (за необхідністю).

2. Який алгоритм називається лінійним?
   Лінійний алгоритм — це алгоритм, у якому всі дії виконуються послідовно, одна за одною, без розгалужень і циклів.

3. Запишіть усі можливі оператори, які дозволять значення змінної a збільшити на одиницю:
   - a = a + 1;
   - a += 1;
   - a++; (постфіксний інкремент)
   - ++a; (префіксний інкремент)

4. Що таке “проект”?
   Проект — це сукупність файлів (вихідних кодів, конфігурацій), необхідних для створення виконуваної програми.

5. Який порядок створення нового порожнього проекту?
   1. Створити папку проекту.
   2. Створити необхідні файли (вихідний код, Makefile, документацію).
   3. Додати файли до системи контролю версій (Git).

6. Як виконати збірку програми?
   Збірка виконується за допомогою Makefile. У терміналі потрібно виконати команду:
   make

7. Який порядок додавання нового файлу до проекту?
   1. Створити файл у відповідній папці.
   2. Додати файл до Makefile, якщо це необхідно.
   3. Додати файл до індексу Git:
      git add <ім'я файлу>

8. Як виправити помилку у програмі?
   1. Ознайомитись із повідомленням про помилку компілятора.
   2. Перейти до відповідного файлу і рядка.
   3. Внести необхідні зміни у код.
   4. Зберегти зміни та повторити збірку.

9. Як поставити програму на виконання?
   Після збірки програма запускається командою:
   ./program

10. Чим консольні програми відрізняються від віконних програм?
    - Консольні програми працюють через командний рядок і мають текстовий інтерфейс.
    - Віконні програми мають графічний інтерфейс із кнопками, меню тощо.

11. Чим змінна відрізняється від константи?
    - Змінна — це область пам'яті, значення якої можна змінювати під час виконання програми.
    - Константа — це фіксоване значення, яке не можна змінити після оголошення.

12. Чим відрізняється префіксний запис операції інкременту від постфіксного запису?
    - Префіксний запис (++a) збільшує значення змінної на одиницю перед використанням.
    - Постфіксний запис (a++) збільшує значення змінної після її використання.
